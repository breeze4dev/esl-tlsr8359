## 电子价签 Web 设置工具

本仓库提供一个基于浏览器的工具，用于通过蓝牙/Web 串口对采用 TLSR8359 芯片的汉朔 2.13 寸三色电子价签进行图片传输、时间/日历设置以及固件升级。

### 功能概览

- **蓝牙连接**：搜索名为 `ESL` 的设备并连接。
- **设备控制**：支持模式切换、时间同步、设置时钟样式，选择日期、自定义命令发送等。
- **图片传输**：加载本地图片，支持裁剪、画笔/文字编辑、多种抖动算法，转换后发送到价签显示。
- **固件升级**：
  - `uart_flasher.html`：通过串口进行固件刷写。
  - `ATC_TLSR_PaperBLEcontrol.html`：通过蓝牙进行 OTA 升级。

### 使用方式

1. 访问地址:<https://breeze4dev.github.io/esl-tlsr8359>
2. 打开页面后，点击“连接”按钮，选择名称以 `ESL` 开头的设备建立蓝牙连接。
3. 根据页面分区使用日历/时钟设置、图片传输、或固件升级功能。

> 如需了解底层固件及更多细节，可参考页面中链接的上游开源项目。
